# VIP ტერმინოლოგიის განახლება

პროექტში ერთიანი ტერმინოლოგია გადავიყვანე ამ ვარიანტზე:
- VIP
- VIP განთავსება
- VIP განცხადებები

ძირითადი ცვლილებები:
- ღილაკები და ნავიგაცია: „გაძლიერება“ -> „VIP განთავსება"
- შეკვეთები/ბილინგი: „გაძლიერების შეკვეთები“ -> „VIP განთავსების შეკვეთები"
- სტატუსები და სექციები: „გაძლიერებული“ -> „VIP“ / „VIP განცხადებები"
- listing promote გვერდი: „განცხადების გაძლიერება“ -> „განცხადების VIP განთავსება"
